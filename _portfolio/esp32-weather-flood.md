## Overview

An ESP32-based IoT weather and flood monitoring system that measures temperature, humidity, rainfall, and water level in real time. The system establishes a baseline distance using an ultrasonic sensor, continuously monitors flood risk, and displays environmental data on an OLED screen. Sensor data is transmitted wirelessly via Bluetooth Low Energy (BLE) to a responsive web dashboard.

## Features

- Real-time monitoring of temperature, humidity, rainfall, and water level
- Flood detection using ultrasonic sensor
- OLED display for local monitoring
- Bluetooth Low Energy (BLE) communication
- Interactive web dashboard
- Machine Learning-based prediction using Quadratic Regression
- Early warning for abnormal environmental conditions
